# README
###Market Melody
#####Hear your Profile
A conceptual melodizer meant for audio-visualization of data.

Hear your Portfolio.

built with:
-ruby 2.6.5  
-React.js   
-Google charts   
-Tone.js   
-Carrierwave    
-AlphaVantage API   

To run the database   
-bundle exec rake db:create   
-bundle exec rake db:migrate   
-bundle exec rake db:seed   
-bundle exec rspec
